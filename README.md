# 🎴 ColorVault

**ColorVault** is a browser-based collectible card game about colors — pull, collect, trade, and climb the leaderboard. No backend, no build step: everything runs in a single HTML file using `localStorage`.

🇷🇺 [Русская версия ниже](#-colorvault-1)

---

## ✨ Features

- **Color pulls** — generate random colors, each scored for rarity (Common → Mythic) based on saturation, lightness, and hue purity
- **Pity system** — guaranteed rarity boosts the longer you go without a good pull
- **Collection view** — browse cards by color family or rarity
- **Trading** — send and accept trade offers with other players
- **Leaderboard** — compete by total collection value
- **Persistent accounts** — login/register system backed by `localStorage`
- **Cyberpunk/gold UI** — custom fonts (Bebas Neue, DM Mono, Manrope) and a dark, neon-accented theme

## 🚀 Play

Just open `index.html` in any modern browser — no installation or server required.

Or enable **GitHub Pages** for this repo (Settings → Pages → deploy from `main` branch) to play it online.

## 🛠️ Tech

- Vanilla JavaScript, HTML, CSS — no frameworks, no build tools
- Data persistence via browser `localStorage`
- Single-file architecture for easy hosting/sharing

# 🎴 ColorVault (RU)

**ColorVault** — браузерная коллекционная карточная игра про цвета: открывай паки, собирай коллекцию, торгуй с другими игроками и поднимайся в таблице лидеров. Работает без сервера и сборки — всё в одном HTML-файле, данные хранятся в `localStorage`.

## ✨ Возможности

- **Открытие цветов** — случайная генерация цветов с расчётом редкости (Обычный → Мифический) по насыщенности, светлоте и чистоте оттенка
- **Система пити (pity)** — гарантированное повышение редкости, если долго не выпадает ценная карта
- **Коллекция** — фильтрация по цветовым семьям и редкости
- **Торговля** — отправка и принятие торговых предложений между игроками
- **Таблица лидеров** — рейтинг по общей стоимости коллекции
- **Аккаунты** — вход/регистрация с сохранением в `localStorage`
- **Киберпанк/золотой стиль** — кастомные шрифты (Bebas Neue, DM Mono, Manrope) и тёмная неоновая тема

## 🚀 Как играть

Просто открой `index.html` в любом современном браузере — установка и сервер не нужны.

Либо включи **GitHub Pages** для этого репозитория (Settings → Pages → deploy from `main` branch), чтобы играть онлайн.

## 🛠️ Технологии

- Чистый JavaScript, HTML, CSS — без фреймворков и сборщиков
- Хранение данных через `localStorage` браузера
- Однофайловая архитектура для лёгкого хостинга
